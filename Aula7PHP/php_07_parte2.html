3. Salvando os dados

Passos:

  1. Conectar ao banco de dados.
  2. Receber os dados do formulário.
  3. Preparar o comando SQL.
  4. Executar o comando SQL.
  5. Fechar a conexão com o banco.

3.1. Conexão com o banco de dados

Usando new PDO(): 

$servidor = 'localhost';
$banco = 'receitas';
$usuario = 'root';
$senha = '';

$conexao = new PDO("mysql:host=$servidor;dbname=$banco", $usuario, $senha);

3.2. Recebendo dados

Os dados chegam em $_GET de acordo com o name de cada dado. 

<!-- $_GET['nome'] -->
<input type="text" name="nome"><br>

<!-- $_GET['quantidade'] -->
<input type="text" name="quantidade"><br>

3.3. Preparar e executar o comando SQL.

Vamos usar phpmyadmin para descobrir o comando SQL e vamos usar prepare() e execute() para inserir os dados. 

$comando = $conexao->prepare($codigoSQL);

$resultado = $comando->execute(array('nome' => $_GET['nome'], 'qtd' => $_GET['quantidade']));

if($resultado) {
  echo "Comando executado!";
} else {
  echo "Erro ao executar o comando!";
}

3.4. Fechar a conexão

$conexao = null;

3.5. Página final

<?php

// conexao
$servidor = 'localhost';
$banco = 'receitas';
$usuario = 'root';
$senha = '';

$conexao = new PDO("mysql:host=$servidor;dbname=$banco", $usuario, $senha);

echo "Conectado!<br>";

echo "Recebido: <br>";
echo $_GET['nome'];
echo "<br>";
echo $_GET['quantidade'];

$codigoSQL = "INSERT INTO `ingredientes` (`id`, `nome`, `quantidade`) VALUES (NULL, :nm, :qtd)";

try {
    $comando = $conexao->prepare($codigoSQL);

    $resultado = $comando->execute(array('nm' => $_GET['nome'], 'qtd' => $_GET['quantidade']));

    if($resultado) {
	echo "Comando executado!";
    } else {
	echo "Erro ao executar o comando!";
    }
} catch (Exception $e) {
    echo "Erro $e";
}

$conexao = null;

?>

4.1. Exportando banco de dados

Passos para exportar o banco de dados com suas tabelas:

  1. clique no nome do banco.
  2. acesse a aba Exportar.
  3. marque o método de exportação: Personalizada.
  4. marque adicionar comando CREATE DATABASE / USE.
  5. aperte o botão Exportar.

O arquivo de criação do banco é recebido por download.
